# kalambur.nl

## Technologies

**Core**
- Next.js 16 (App Router) + React 19
- TypeScript
- pnpm (package manager)

**Styling**
- Tailwind CSS v4
- shadcn/ui (Radix UI primitives, class-variance-authority, tailwind-merge, lucide-react icons)
- tw-animate-css

**Database**
- Neon serverless Postgres (`@neondatabase/serverless`)

**Analytics**
- Vercel Analytics

## Deployment 

This project is deployed to Vercel.
